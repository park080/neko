웹 프로그래밍 공부하기

  1. 자신에게 맞는 에디터 찾기
  2. 시작하기에 앞서
  3. 태그정리
  • !doctype html 이문서는 html로작성되었다고 알려주는태그
  • html 가장 큰범위의 태그
  • head 제목부분을 특정해주는태그
  • title 탭에보이는 제목
  • meta 브라우저에게 특정코드로 읽으라고 알려주는 태그
  • body 본문을 나타내주는 태그
  • a는 링크를 걸어주는태그 a href="주소"
    새탭으로 열리게할경우 뒤에 추가로 target="_blank"
    마우스를 대었을때 어디로이동할건지 추가로알리고싶은경우 title="정보"
  • u 밑줄
  • b는 단순한 글자를 굵게만드는것 strong는 실제로 강조하는것
  • h1~6 제목크기
  • br 한줄을 띄는것/로 닫을 필요가 없다
  • p 문단 만들기
  • style="margin-top??px"를사용시 위아래간격을 조정할수있다
    top 위치를나타내는말로 left,right,top,bottom이 있다
  • style="width:???px"를 사용시 좌우여백을 설정할수있다
  • span(inline레벨단위 영역설정)or div(box레벨영역설정) style="font-size:??px"사용시 글씨크기를 설정할수있다.
  • img뒤에 src="같은디렉토리안에있는 파일의 이름" width="???"를 추가시키는것으로 사진을 추가할 수 있다
  • li는 목차를 나타낼때 사용
  • ul태그는 li를 묶을때 사용할수있다
  • ol태그는 리스트앞에 순서대로 숫자를부여해준다
  • !-- .... --는 이사이에있는 명령들을 무시하게해준다
  • head에 들어가는 style의경우 css문법으로 해석하라고 말해주는 명령어이다
  • style은 html의 속성으로도 사용할수있다 이때 head와 중복되는경우 속성을따른다
  • 속성을 여러개쓸때 ;으로 구분해준다
  • /*head에 스타일부분에들어가는 이것은 css의 주석*/
  • h1처럼 화면전체의 박스크기를가지는것을 block level element, 자신의 크기만큼만 박스크기를 가지는것을 inline element라고부른다
